
## 制作儿童英语音乐启蒙视频

```
过程

1. 生成歌词和绘图

1.1 你是一名非常专业的内容创作者，这是一个爆款英语启蒙类的音乐视频，分析这个视频能爆火的原因，总结这个视频的文案特点，以及视频中音乐的风格

1.2 根据刚刚总结的文案特点以及爆火原因，写一篇类似的歌曲歌词，这首歌的主题是给儿童介绍26个英文字母，要求歌词生动有趣，能够吸引小孩的注意

1.3 现在给这首歌生成一个非常详细的场景系列(至少26个场景)要求每个场景都符合当前这首歌正在介绍的动物或者物品

1.4 创建非常详细的绘图 prompt，用于为每个场景生成图片，图片要采用适合小孩观看的风格，这些 prompt 我将用于在 Midjourey 中生成图片

2. 生成创作歌曲的 Prompt

2.1 YouTube 上找童谣歌曲频道，找适合的连接

2.2 SONOTELLER 粘贴连接，稍等会出现对这个视频的分析，或者用 Suno Prompt

3. 生成歌曲：Suno，用 1 中的歌词，用 2 中的 Style of Music

4. 生成歌曲配图，tensor.art，推荐用 Disney 模型，NUKE 子类，

5. 视频剪辑：CapCut，比较费时间，调整视频，加字幕

```
